## jQuery
<aside class="notes"></aside>

* jquery.com - A JavaScript library that makes DOM manipulation simple.
* Cross browser - works the same in all* browsers. 
* Allows:
  * Document traversal
  * CSS Manipulation
  * Event Handling
  * Animation
  * and more!



## Adding jQuery to your website
<aside class="notes"></aside>

<br>

__OR__

<br>
